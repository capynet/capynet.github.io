GIT: Mostrar gráfico de ramas en consola

  • capynet
  • 10 Mar, 2015
  • Tip
  • git

Para conseguir que la consola te muestre el gráfico e ramas (tree view en rigor) en la consola haz lo siguiente:

1
nano ~/.gitconfig

y al finar de lo que haya dentro del archivo pega esto:

1
2
[alias]
glog = log --all --graph --decorate --oneline

Listo, ya podés hacer

1
git glog

NOTA: glog = “graphic log” Chau!

Instalar Skype en Debian 7

  • capynet
  • 1 Feb, 2014
  • Linux, Tip
  • Debian, Linux

Problema: Skype tiene un paquete .deb multiarquitectura… pero en Debian solo podés instalarlo en 32 bits. Solución:

1
sudo dpkg --add-architecture i386 && sudo apt-get update

Ya podés instalar el .deb sin problemas. Chau!

Linux: Convertir imágenes de png a jpg desde la consola

  • capynet
  • 13 May, 2013
  • Linux, Tip
  • Linux, Programas
  • 4 comments

Una vez mas esto es una nota mental mas que un tutorial: Si no tenés instalado imagemagick:

1
sudo apt-get install imagemagick

Luego para convertir usamos el comando convert:

1
convert imagen.png imagen.jpg

Y para convertir todas las imagenes en un directorio:

1
convert *.png *.jpg

Chau!

Añadir tu usuario de linux al grupo www-data

  • capynet
  • 1 Mar, 2013
  • Linux, Tip
  • Linux, Ubuntu
  • 3 comments

Cuando instalo un LAMP en algún Linux suelo añadir mi usuario (capy) al grupo www-data, cambio el directorio www al grupo www-data y le doy permisos de escritura a los usuarios del grupo para garantizarme que no vaya a tener problemas de permisos y cosas raras. Lo siguiente hay que hacerlo una sola vez:

1
2
3
4
sudo chgrp -R www-data /var/www
sudo chmod -R g+w /var/www
sudo find /var/www -type d -exec chmod 2775 {} \;
sudo find /var/www -type f -exec chmod ug+rw {} \;

Si habilitan el modulo intl en Xampp puede que les de error al no encontrar la librería “icuuc46.dll“.  La solución en este caso nos la da Cyn Wong: Ir a [xampp_path]/php/ y copiar estos tres archivos: icudt46.dll icuin46.dll icuuc46.dll Pegalos en [xampp_path]/apache/bin y reinicia, y vas a ver como ya no hay problemas.
