Snippet: Verificar si una fecha está dentro de un rango de fechas en PHP.

  • capynet
  • 3 Jun, 2010
  • PHP, Programación, Tip
  • PHP, Programación, snippet
  • 10 comments

La saqué de la documentación de PHP creo recordar, y a alguien le puede ser útil 😉

1
2
< ?php /** * Verifica que una fecha esté dentro del rango de fechas establecidas * @param $start_date fecha de inicio * @param $end_date fecha final * @param $evaluame fecha a comparar * @return true si esta en el rango, false si no lo está */ function check_in_range($start_date, $end_date, $evaluame) { $start_ts = strtotime($start_date); $end_ts = strtotime($end_date); $user_ts = strtotime($evaluame); return (($user_ts >= $start_ts) && ($user_ts <= $end_ts));
}

    echo "estas en rango";

    echo "fuera de rango";

Aplicar sombra sólida a un div con CSS en Internet Explorer

  • capynet
  • 20 May, 2010
  • CSS y Maquetación
  • CSS, snippet
  • 2 comments

Agreguen la siguiente línea en el selector del div:

1
filter: progid:DXImageTransform.Microsoft.DropShadow ( Color = '#ececec', OffX = '5', OffY = '5' );

NOTAS: OffX : cuanto desplazamos horizontalmente la sombra. OffY : cuanto desplazamos verticalmente la sombra. Muchísimo ojo con dejar espacios entre “progid:DXImageTransform.Microsoft.DropShadow” porque deja de funcionar.
