$.slideToggleMinHeight()

jQuery.slideToggleMinHeight( [minHeight], [speed], [easeEffect], [callback])

Download

minHeight: A number determining the min height. Default value 50px if any specified.
speed: A number determining how long the animation will run. Default is 500 ms.
easeEffect: A string indicating which easing function to use for the transition. Available transitions in http://gsgd.co.uk/sandbox/jquery/easing/
callback: A function to call once the animation is complete. the method sends the jQuery object and action performed.

Example

  • Make me slide with default options! 
    $(".divToSlide").slideToggleMinHeight(100);
  • Make me slide with 2 sec! 
    $(".divToSlide").slideToggleMinHeight(
            100,
            2000
        );
  • Make me slide with 2 sec and "easeOutBounce"! 
    $(".divToSlide").slideToggleMinHeight(
            100,
            2000,
            "easeOutBounce"
        );
  • Make me slide with 2 sec, "easeOutBounce" and my callback! 
    $(".divToSlide").slideToggleMinHeight(
        100,
        2000,
        "easeOutBounce",
        dummyCallback
);

dummyCallback = function(obj, actionPerformed) {
    alert('Action "'
    + actionPerformed +
    '" finished over ".' +
    obj.attr("class") +
    '" selector');
}
